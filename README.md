## usage

1. Clone the repo

    ```
    git clone https://github.com/sgbj/cool-af-sans.git
    ```
    
2. Install the dependencies

    ```
    cd cool-af-sans
    npm install
    ```
    
3. Add custom SVGs (1000 x 1000 recommended) to `src/icons`
4. Build the font (ttf, woff, eot)

    ````
    npm run build
    ````
    
5. Check out the results in `dist`
